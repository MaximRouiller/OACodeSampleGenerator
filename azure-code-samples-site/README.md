# Azure REST API Code Samples Site

Built with [Gatsby](https://www.gatsbyjs.com/) static site generator and [Chakra-UI](https://chakra-ui.com/) component library.

## Quick start

1.  **Install packages.**

    ```shell
    npm install
    ```

2.  **Develop.**

    ```shell
    npm run develop
    ```

3.  **View the running development site at:**

    http://localhost:8000
